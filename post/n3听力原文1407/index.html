問題１

  1. 女の学生と先生が話しています。女の学生はこの後、何をしますか。

    • 女：あのう、先生、来週の発表の資料、見ていただけましたか。
    • 男：ああ、あれね、見ましたよ。前に言った表の数字の間違いもなおってましたし、よかったと思いますよ。
    • 女：他になおしたほうがいいところはありませんでしたか。
    • 男：そうですね、全体的にもうすこし字を大きくしたらどうですか。そのほうが見やすいと思いますよ。
    • 女：わかりました。
    • 男：それ以外は、グラフも分かりやすかったし、説明が足りないところも特にありませんね。

  2. 大学で男の学生と女の学生が話しています。女の学生は何をしなければなりませんか。

    • 男：木村さん、今度のゼミの旅行、木村さんが係だよね。どうなってる？泊まるところは決まった？
    • 女：はい、去年と同じホテルを予約しました。会議室も取っておきました。
    • 男：そう、あそこの会議室、広くていいよね。去年もバスで行くの？
    • 女：はい、一台予約しました。
    • 男：バス、去年は小さくて大変だったから、今年は少し大きいのを借りたほうがいいと思うよ。
    • 女：そうですね。じゃ、大きいのに変えます。
    • 男：うん、それから、先生にはにってい、もうお伝えしてあるよね。
    • 女：もちろんです。

  3. 大学で男の学生と女の学生が話しています。男の学生はこの後どうしますか。

    • 男：今日の授業の発表、すごく面白かったよ。
    • 女：ほんとう。ありがとう。
    • 男：参考にしていた本、どこで見っけたの。
    • 女：伊藤せんぱいに借りたんだ。この大学の図書館にはなかったし、このあたりの本屋にも売ってなかったんだ。
    • 男：僕も借りたいな。伊藤せんぱいに電話して聞いてみよう。
    • 女：あっ、いま、松田さんが借りて読んでるよ。次も借りたい人がいるって言ってたから、読めるの、ずいぶん後になっちゃうね。
    • 男：そうなんだ。早く読みたいよ。
    • 女：たしか森先生が持っていらっしゃると思うから、聞いてみたら。順番を待ってたら、一ヶ月はかかっちゃいそうだし。さっき研究室にいらっしゃるのを見たから、今から行ってきたら。
    • 男：そうだね。そうしてみるよ。

  4. 女の人と男の人が電話で話しています。女の人はこの後まず、何をしなければなりませんか。

    • 女：はい、田中印刷です。
    • 男：もしもし、桜デパートの佐藤と申しますが、林さんはいらっしゃいますか。
    • 女：申し訳ありません。林はいま外出中なんですが。
    • 男：では、伝言をお願いできますか。お願いしていたパンフレットができたと伺ったんですけど、早くみたいので、ファックスで送ってほしいとお伝えください。
    • 女：桜デパート様のパンフレットですね。林から聞いております。
    • 男：そうですか。では、申し訳ありませんが、ファックスお願いできますか。
    • 女：承知しました、では、さっそく。
    • 男：あ、それから、パンフレットは百部お願いしているんですが、全部郵送していただくことになっていますよね。
    • 女：はい。
    • 男：その件なんですが、なるべく早くほしいので、すみませんが、明日持ってきていただけませんか。
    • 女：わかりました。

  5. ギター教室で男の人と受付の人が話しています。男の人は来週の火曜日に、何を持ってこなければなりませんか。

    • 男：あのう、すみません。ギター習いたいんですけど。
    • 女：はい。では、こちらの教室についてご説明します。どのクラスもレッスンは週に1回で、レッスン料が一ヶ月6,000円です。あと、会員になる時に、4,000円いただいているんですが、今ちょうど特別期間中なので、来週の火曜日までに会員になっていたたければ、4,000円は無料になります。
    • 男：そうですか、できねば今日申し込みたいんですが、お金は後でもいいですか。
    • 女：はい。火曜までは払っていただはれば結構です。
    • 男：じゃ、お願いします。火曜日なら来られますから。あのう、その日からレッセン受けられますか。
    • 女：はい。あ、レッセンの時にはギターが必要ですが、持っていらっしゃいますか。予約していただければ、教室にあるものも、使っていただけますよ。
    • 男：あ、ギター持ってるんで、自分のを使います。
    • 女：それでは、火曜日、お待ちしています。

  6. 教室で先生が話しています。明日の朝、学生はどうしますか。

    • 先生：明日の授業について連絡します。天気予報によると、明日の朝は大雪になるそうです。それで、バスや電車が動かない場合は、学校は休みになります。明日授業があるかどうかは朝6時までに学校のホームページでお知らせします。皆さんにメールや電話で連絡しませんから、かならず確認してください。学校には直接電話をしないでください。また、今日の午後5時から明日の朝8時半まで学校の中には入れません。今日は早く帰宅してください。

問題２

  1. 男の学生と女の学生が話しています。男の学生は、高校生活で、何が一番いい思い出だったと言っていますか。男の学生です。

    • 男：もうすぐ卒業だね。この三年間、いろんなことがあったね。
    • 女：山本君は、高校生活で、何が一番いい思い出だった。
    • 男：うん、そうだな。すもうぶに入って、一生懸命練習したこと、アルバイトをして、自分でお金をかせいだこと。いろいろあるよ。
    • 女：そうなんだ。
    • 男：森さんは。
    • 女：私は夏休みに友達と旅行をしたことお楽しかったけど、やっぱりクラスで映画を作ったことだよ。
    • 男：ああ、それがあったね。うん、僕もそれが一番いい思い出だった。

  2. 会社で、女の人と男の人が話しています。男の人は、新しい社員はどんな人だといっていますか。

    • 女：今度、営業部に新しい人が入ったんだって。
    • 男：うん。営業の経験はないらしいんだけど、明るいし、営業に向いているのかなって思うよ。佐藤さんっていうんだけど、元気な人だよ。
    • 女：そう。
    • 男：とにかく、声が大きいんだよ。学生時代に、スポーツをやってたんだって。毎朝、大きな声で、「おはようございます」って部屋に入ってくるんだ。
    • 女：そうなんだ。
    • 男：商品についての知識はまだまだだろうけど、まじめだし、勉強するば、すぐにいい仕事ができるようになると思うよ。僕も先輩として、ていねいに教えてあげたいって思っているんだ。
    • 女：いい人が来てくれてよかったね。

  3. 女の人と男の人が話しています。男の人は、どうして子供と料理をすると、大変だと言っていますか。

    • 女：山田さん、お子さん、何歳になりましたか。
    • 男：十歳になったよ。最近は料理に興味を持ちはじめてね、僕も料理が好きだから、よく一緒に作るんだけど、大変だよ。
    • 女：ああ、お子さんと一緒だと、時間、かかっちゃいますよね。それに、包丁で手を切ったりして、けがをしないか心配でしょう。
    • 男：いやあ、そうじゃないんだ。本の通りに作ろうとして、香辛料や調味料、材料とか、全部買いたがるから。結構お金がかかっちゃうんだ。
    • 女：ああ、そうなんですか。家は夫が料理すると、台所を汚すから、困っちゃうんですよ。

  4. バス停で女の人と男の人が話しています。二人はこの後、まず何をしますか。

    • 女：ああ、バスが。
    • 男：ああ、行っちゃった。この辺、バスの数、少ないから、次のバスまで一時間もあるよ。こんな寒いとこで一時間も待ちたくないな。
    • 女：タクシーも全然来そうにないですね。
    • 男：歩いたら、駅まで一時間以上かかるだろうな。
    • 女：へえ。
    • 男：あっ、あそこに食堂がある。あそこに入ろう。
    • 女：はい。食堂から、タクシーを呼びましょう。
    • 男：いや、なんか食べてるうちに、ちょうどバスの時間になるよ。
    • 女：それもそうですね。タクシー呼んでも、ここに到着するまでに、結構時間かかりそうですしね。

  5. 薬屋で女の人と店員が話しています。女の人はどの薬を買いますか。

    • 女：すみません、子供が虫に刺されて、薬を探しているんですが、とてもかゆいみたいなんです。
    • 店員：子供用の薬ですね。お子さんはおいくつですか。
    • 女：四歳と一歳、二人なんですが。
    • 店員：ああ、一歳のお子さんだと使っていただけるものがクリームタイプのものしかないんですよ。
    • 女：じゃあ、それください。あのう、四歳の子もそれ使えますか。
    • 店員：ええ、使えますが、ちょっと効果が弱いんですよね。四歳のお子さんだと、液体のタイプとはるタイプがあります。はるタイプなら、かゆいところをかけないので、傷が広がりませんよ。
    • 女：じゃ、四歳の子のはそれにします。いつもかいてしまうんで、困ってたんです。

  6. 電話で、女の人と車を修理する工場の人が話しています。女の人はどうして、電話をしましたか。

    • 女：もしもし。
    • 男：はい、桜自動車です。
    • 女：あのう、先週車のハンドルの具合が悪くて、修理してもらった鈴木ですが。
    • 男：はい、いつもありがとうございます。どうされましたか。
    • 女：あのう、さっきちょっと車に乗ったんですが、エンジンの調子が、変な音がしているんです。
    • 男：そうですか。
    • 女：ハンドルの修理をお願いしたとき、ついでにエンジンも一緒に見てもらったんですけど、修理代にはエンジンのチェックの料金も入っていました。
    • 男：申し訳ありません、もう一度チェックしますので、今から車をとりにうかがってもよろしいでしょうか。
    • 女：ええ、お願いします。
    • 男：ほかにも気になるところはございますか、一緒に見ておきますが。ブレーキはどうですか。
    • 女：あ、問題ありません。
    • 男：わかりました、では、伺いますので、よろしくお願いします。

問題３

1. 教室で先生が話しています。先生は主に何について話していますか

  • 1.明日の持ち物
  • 2.明日の天気
  • 3.山登りの仕方
  • 4.遠足の予定
  • 男：明日の遠足ですが、お弁当は途中買う店がないので、必ずもってきて下さい。それから、山の天気はとても変わりやすいので、かさやレインコート、帽子、上着などを忘れずに準備して下さい。山登りがしやすいように、荷物は一つにしましょう。それから、遠足の予定の紙がありますね。こちらも忘れないように。

2. 大学で、男の学生と女の学生が話しています。男の学生が言いたいことは何ですか

  • 1.話し合いの時間を変えてほしい
  • 2.話し合いの日を変えてほしい
  • 3.話し合いに遅れる
  • 4.話し合いに出られない
  • 男：林さん、あさってのゼミの話し合いのことなんだけど。
  • 女：ああ、来月の、調査の？
  • 男：うん、あれ、三時から五時までだったよね。実は、ちょうど同じ時間に、中村先生に研究室に呼ばれちゃってね。夜遅くまでかかりそうなんだよ。
  • 女：へえ、係りを決めるから、全員出席ってことになっているよ。
  • 男：ごめん。今から他の日に変えてもらうのは無理だろうし、係りはなんでもするから、決まったら教えて。
  • 女：うん。わかった。

3. テレビで、男の人が話しています。男の人は何について話していますか

  • 1.この米の味がいい理由
  • 2.この米が有名になった理由
  • 3.この米を消費者に直接届ける理由
  • 4.日本の米の消費量が多い理由
  • 男：最近、人気をあつめている「米ピカリ」という新しいお米。このお米はインターネットで注文を受けて、消費者に直接届ける方法で日本全国に売られています。テレビコマーシャルなどの宣伝もなく、ここまで名前が広く知られるようになったのは、生産者の人たちが美味しいお米を作る努力をされたことはもちろんですが、買った人々がインターネットでこのお米を食べた感想を伝えたことが大きいようです。

問題４

1. 結婚するので、部長に知らせます。何と言いますか

  • 1.結婚されるそうですよ。
  • 2.結婚式には、必ず伺います。
  • 3.今度結婚することになりました。

2. エレベーターが動いていません。後から来た人に何と言いますか

  • 1.お先に失礼します。
  • 2.動かないでくださいね。
  • 3.今、使えないようですよ。

3. 友達のコートのボタンがありません。友達に何と言いますか

  • 1.ボタン、取れてるよ。
  • 2.ボタン、これなんじゃない。
  • 3.ボタン、もらっとくね。

4. 図書館です。イヤホンの音がうるさいかどうか聞きたいです。隣の人に何と言いますか

  • 1.音、大きくしましょうか？
  • 2.音、気になりますか？
  • 3.音、きちんと聞こえますか？

問題５

1. いらっしゃいませ。何名様ですか

  • 1.中山と申します。
  • 2.もう一人来るので、3人です。
  • 3.一人千円ずつです。

2. 木村さん、よろしければ、この後食事でも

  • 1.ええ、行きましょう。
  • 2.どうもご馳走様でした。
  • 3.どうぞ遠慮なさらずに。

3. あのう、すみません。このペン、ちょっとお借りできますか

  • 1.あっ、お使いください。
  • 2.すぐお返しします。
  • 3.たいへん結構です。

4. 田中さん、この書類なんですが、見ていただけませんか

  • 1.いいですが、何の書類ですか。
  • 2.一緒に見に行きませんか。
  • 3.今ご覧になっています。

5. 駅前にできたラーメン屋に行ってみない？おいしいらしいよ

  • 1.どうしても行けないの？
  • 2.私も行きたいと思ってたんだ。
  • 3.食べに行ってどうだった？

6. 佐藤君のおかげで、会議の準備早めに終わったよ

  • 1.また手伝うから、いつでも言って。
  • 2.手伝ってくれてありがとう。
  • 3.何から始めればいい？

7. 隣のクラスの山田君、マラソンで次のオリンピックに出ることになったんだって

  • 1.そうなるといいね。
  • 2.あと少しだったのにね。
  • 3.みんなで応援しなくちゃね。

8. 松本さん、部長への主張の報告はもう済んでるよね

  • 1.えっ？報告しなくてもいいんですか。
  • 2.すみません、朝からずっと会議に出てたので。
  • 3.おとといお戻りになりました。

9. 先輩、きのう面接受けたんですけど、予想外のこと、質問されちゃったんです

  • 1.予想があたっちゃったんだ。
  • 2.きちんと質問できて、よかったね。
  • 3.僕も経験あるよ。あわてるようね。
