問題１

  1. 市役所で、男の人と女の人が話しています。男の人はこのあと何をしなければなりませんか。

    • 男：課長、来週行う経済セミナーの件ですが、さっき見てみたら、当日使う予定のマイクが一本壊れてたんです。
    • 女：そう。確か、市民館に新しいのがあるはずだから、借りてきます。ちょうど今から別の用で行くところですから。
    • 男：ありがとうございます。
    • 女：あ、当日配る資料だけど、講演して頂く森先生に前に送ってもらったものでいいか確認しておいてくれますか。
    • 男：分かりました。すぐメールします。
    • 女：いいえ、メールはいつ見てもらえるか分からないから、電話でお願いします。
    • 男：あ、はい。
    • 女：それから、スピーカーも一応音が出るか確認して、だめだったら市民館に借りに行ってくださいね。
    • 男：それは、さっきマイクの確認と一緒にしました。
    • 女：そうですか。

  2. 会社で、男の人と女の人が話しています。女の人はこのあとまず、何をしなければなりませんか。

    • 男：鈴木さん、来月田中部長が海外の支店に移られることになったの、聞いた？
    • 女：はい。
    • 男：部長にはいろいろお世話になったし、最後に食事会をしようと思うんだけど。その時、プレゼントと一緒にカードを渡したいんだ。悪いんだけど、ガード買って、皆に回してメッセージを書いてもらって。
    • 女：わかりました。
    • 男：人数多くて時間かかるかもしれないから、あしたからでも回し始めないとね。プレゼントは私が何か用意しておくよ。
    • 女：はい。
    • 男：それから、食事会の場所だね。
    • 女：私、いいレストラン知ってるので、日が決まったら、予約しておきます。
    • 男：ありがとう。来週部長が出張から戻られたら、ご予定を伺ってみるよ。
    • 女：分かりました。

  3. 日本語学校の教室で先生が話しています。学生はこのあとまず、何をしなければなりませんか。

    • 今日は中村先生が急な用でいらっしゃらないので、かわりに私が来ました。この時間は、一人ずつ自分の趣味について発表をする予定でしたが、それは明日の授業でしてもらいます。いつもやっている漢字テストは、今日はありません。えー、昨日の授業で物語を読みましたね。原稿用紙を配りますから、感想を400字ぐらいで書いてください。それから、昨日宿題がでましたね、きょうかしょの練習問題。それは、あした中村先生に出してください。じゃ、始めましょう。

  4. スーパーで、店長と女の店員が話しています。女の店員はこのあとまず何をしなければなりませんか。

    • 男：山田さん、おはよう。ちょっといい？
    • 女：はい、店長。
    • 男：新しくバイトに入った林君のことなんだけど、山田さんが指導係になって、いろいろ教えてあげてほしいんだ。
    • 女：え？私がですか。
    • 男：うん、よろしく頼むよ。いま林君には店の前を掃除してもらってるんだけど、もう終わるころだから、店のどこにどんな商品が並んでるのか教えてあげて。
    • 女：はい。
    • 男：それから、午後はレジの仕事の説明をお願い。彼にはできるだけはやくレジを覚えてもらいたいんだ。
    • 女：わかりました。閉店後の店の中の確認も一緒にしたほうがいいですか。お客さんのおとしもののかいしゅうとか。
    • 男：そうだね。それもお願いするよ。

  5. ゴルフ教室で、男の人と受付の人が話しています。男の人はこのあと、どのクラスに申込みますか。

    • 男：あのう、すみません。ゴルフのレッスンに申込みたいんですが。どのクラスにすればいいのかよく分からなくて。
    • 女：はい。クラスは全部で四つありますが。ゴルフは初めてですか。
    • 男：いいえ、以前やってました。
    • 女：そうですか。では、入門は始めての方のクラスなので、経験者の方はそれ以外のしょきゅう、ちゅうきゅう、じょうきゅうの三つの中からお選びください。
    • 男：それぞれどんなことをするんでしょうか。
    • 女：しょきゅうは入門クラスで習った基本的な打ち方をしっかり身に付けるクラスです。ちゅうきゅうは基礎を身に付けた人がいろいろな技術を学ぶクラスです。
    • 男：はい。
    • 女：じょうきゅうは、市の大会を目標に頑張っている方多いですよ。
    • 男：市の大会ですか。昔はよく出てたんですけど、しばらくやってないし。
    • 女：あ、すみません、このクラスいま定員がいっぱいになってました。一つしたのクラスから始めてみてはいかがですか。
    • 男：そうですね、そうします。

  6. お城の前で、案内の人が話しています。これからお城の中を見学する人は、入り口で靴を脱いだあと、靴をどうしますか。

    • 皆様、これからお城の中を見学していただきますが、靴は入り口で脱いで、土が落ちないように、ふくろに入れてお持ちください。ふくろは入り口でお渡しします。本日はたいへん混雑しているため、見学通路は一方通行になっております。入り口と出口が違いますので、入り口に靴を置くたながございますが、これは本日はお使いにならないようにしてください。また、脱いだ靴を入り口のすみに置いていかないようにお願いします。

問題２

  1. 携帯電話で女の人と男の人が話しています。二人はどこで会いますか。

    • 女：もしもし、佐藤君？いまどこ？
    • 男：あ、もう駅の北口のかいさつについてるよ。北口のかいさつで合う約束だよね。
    • 女：ごめん。わたし、いま南口にいるんだ。さっき、ハイヒールの踵が折れちゃって…たしか、南口のデパートに靴をなおせるところがあったから、ちょっと寄っていきたいんだけど。
    • 男：そう。じゃ、ぼくも南口のデパートに行こうか？
    • 女：ううん。北口によく行くカフェがあるよね。時間かかるかもしれないから、そこで待ってて。
    • 男：分かった、じゃ、あとで。

  2. 会社で、女の人と男の人が話しています。男の人はどうして今の家に住んでいますか。

    • 女：木村さんって、通勤に2時間かかるんだって？ご両親の家から通ってるの？
    • 男：あ、一人暮らしだよ。
    • 女：うん…一人暮らしなら、会社の近くにも住めるのに、どうして遠くに？会社の近くより、家賃が安いの？
    • 男：いや、家賃はそんなに変わらないけど、部屋がたくさんあって、庭もあるんだ。ぼく、本をたくさん持ってるから、置いておける部屋があると便利で。
    • 女：そうなんだ。庭があるんなら、犬とか伺えるよね。
    • 男：そうだね。でも、出張なんかもあるし、ベットを伺うことは考えてないんだ。

  3. テニス教室で、男の人と女の人が話しています。女の人はどうして最近テニスがうまくなったと言っていますか。

    • 男：山田さん、最近、テニスうまくなったね。
    • 女：ああ、ありがとう。ラケットとか靴とか新しくしたから。
    • 男：ええ？道具を変えると、うまくなるの？
    • 女：はあ、ごめん。それは冗談。うまくなりたくて、何かいい方法がないか、あれこれやってみたの。教室に来て練習する日を増やしてみたり、先生を変えてみたり、それから、自宅でできるトレーニングを始めてみたり…
    • 男：うん、それで？
    • 女：それで、わたしの場合は教室に来る回数を増やすより、うちで1人でできることをやるほうが効果的だって分かったんだ。
    • 男：そっか。なるほど。

  4. 会社で、女の人と男の人が話しています。男の人はどうしてお弁当を持ってきていますか。

    • 女：あ、加藤さん。今日もお弁当？毎朝自分で作ってるの？
    • 男：うん。何を作るか毎日考えなきゃいけないのはちょっと大変だけど。
    • 女：料理作るのを好きなの？
    • 男：いや、あまり好きじゃないんだけど。実は車がほしくて、ずっと貯金してて。外食するとけっこうお金がかかるよね。
    • 女：そうだね。おべんとうだと野菜もたくさんは入れられるから健康にもよさそうだね。
    • 男：あ、健康のことはあんまり考えてなかったけど…そうだね。

  5. 男の人と女の人が話しています。男の人はインターネットで服を買わない一番の理由はなんだと言っていますか。

    • 男：おう、その服、いい色だね。どこで買ったの。
    • 女：これ、インターネットで買ったの。いいでしょう。
    • 男：ああ、インターネットショッピングか。実は、僕、インターネットで服を買ったことないんだ。送料がかかるし、品物がとどくまでに時間がかかるんでしょう。
    • 女：うん、送料がかかるものもあるよ。でも、家にいて、簡単に買い物ができるって、やっぱり便利だと思わない。
    • 男：うん、そうだね。ただ、たいてい注文する前に実物を着てみることはできないんだよね。僕はそれが一番気になるなあ。いままで、なんか問題なかった。
    • 女：うん、サイトを見れば、サイズや色はわかるから、大丈夫だよ。私はそれより、始めて注文したとき、店に名前とか、住所とか教えなきゃならないことが気になったけどね。

  6. 町の体育館の受付で、男の人と係りの人が話しています。男の人は、どうして今日利用の申し込みができませんか。

    • 男：あのう、始めて体育館を利用するんですけど、今日申し込めますか。来月、12名でバスケットボールをしたいんです。
    • 女：代表の方ですね。あのう、この町の方ですか。
    • 男：はい。
    • 女：じゃ、この町に住んでいることがわかる証明書と、あっ、それから、今日、はんこはありますか。
    • 男：はい。ええと、証明書は運転免許証でいいですか。
    • 女：大丈夫です。え…、では、この申請用紙に必要なことを書いてください。利用者の名前も全員分、お願いします。
    • 男：あっ、みんなの分は、いまちょっと…。全員分必要ですか。
    • 女：はい。
    • 男：じゃ、確認して、明日また来ます。
    • 女：すみません。では、この用紙を渡しますから、これに書いて、はんこを押して、お持ちください。利用料金はそのとき現金でお支払いください。よろしくお願いします。

問題３

1. テレビで女のアナウンサーと男の人が話しています。男の人は何についていますか

  • 1.小鳥の世話で大変なこと。
  • 2.小鳥を伺ってよかったこと。
  • 3.小鳥につけた名前の意味。
  • 4.小鳥を上手に伺う方法。
  • 女：佐藤さんは小鳥を伺っていらっしゃるそうですね。
  • 男：ええ、シロという名前なんです。
  • 女：いつごろから伺っていらっしゃるんですか。
  • 男：二年前です。長男が四歳のとき、自分で世話をするという約束で、最初は大変かと思ったんですが、きちんと世話を続けています。生き物を大切にする気持ちを持ってくれているようで、嬉しいです。
  • 女：小さいお子さんでも、世話ができるんですね。
  • 男：ええ、つめを切るとか、そういうことは子供には難しいんで、私がしますけどね。それから、シロを買うようになってから、家族の会話も増えた気がします。
  • 女：そうですか。
  • 男：幼稚園などで、家族の絵を描くときには、必ずシロも入っているんですよ。
  • 女：シロも家族なんですね。
  • 男：そうですね。私たちがシロと名前を呼ぶと、いつもかわいい声で答えるんです。

2. アパートの前で、女の人と隣の部屋の男の人が話しています。女の人が言いていことは何ですか

  • 1.ペーティーに招待してほしい。
  • 2.部屋でパーティーをしないでほしい。
  • 3.外国に行ったときの話を聞いてほしい。
  • 4.夜遅くに騒がないほしい。
  • 女：中村さん、こんばんは。
  • 男：ああ、こんばんは。
  • 女：夕べはお友達たくさん来てたのね。にぎやかでしたね。
  • 男：あ、はい。昨日はパーティーで。
  • 女：でも、皆さん、ちょっと声が大きくて、私、夜なかなか眠れなかったんですよ。
  • 男：ああ、すみませんでした。
  • 女：いえいえ、パーティーって楽しいですよね。私も若いころ友達とよくパーティーしてたから、わかるんです。外国に住んでいた時なんですけどね。パーティーをするのはいいんですけど。ただ、時間がね。
  • 男：はい、気をつけます。外国に住んでいらっしゃったんですか。よかったら、今度話を聞きせてください。
  • 女：ええ、また今度ね。

3. ラジオで女の人が話しています。女の人は何について話していますか

  • 1.冬に病気が流行する理由。
  • 2.インフルエンザの治し方。
  • 3.生活習慣が原因の病気。
  • 4.ウイルスから体を守る方法。
  • 女：寒くなってきましたね。冬は、インフルエンザが流行する季節です。これはウイルスが原因です。インフルエンザは一年中、いつでもかかる可能性がありますが、特に流行する冬は、普段から手洗いや、うがいをきちんと行い、ウイルスを体のなかに取り込まないようにすることが大切です。また、生活習慣を意識し、規則正しい生活を送ると、ウイルスに対する体の力が強くなることが期待できます。

問題４

1. 机の上に友達の傘があります。友達はいま帰るところです。何と言いますか

  • 1.傘、忘れてるよ。
  • 2.傘、持ってこなかったの。
  • 3.傘、ここに置いたら。

2. 写真を撮ってほしいです。近くにいる人に何と言いますか

  • 1.すみません、写真を撮っていただけませんか。
  • 2.あのう、写真を撮ってもよろしいですか。
  • 3.よければ、写真をお撮りしましょうか。

3. おしゃれな店を見つけました。友達を誘って、一緒に入りたいです。何と言いますか

  • 1.ここ、行かなきゃいけない。
  • 2.ここ、この前はいったよね。
  • 3.ここ、ちょっと見ていかない。

4. 辞書を忘れたので、友達の辞書を借りたいです。何と言いますか

  • 1.ぜひ、その辞書使って。
  • 2.その辞書、ちょっと使わせてもらってもいい。
  • 3.その辞書、使ってみるといいよ。

問題5

1. いらっしゃいませ。何名様でしょうか

  • 1.六回目です。
  • 2.六人です。
  • 3.お客様です。

2. 森さん、ちょっと書類の整理を手伝ってほしいんだけど

  • 1.そう、もう手伝わなくてもいいの。
  • 2.ありがとう、じゃ、お願いするね。
  • 3.ああ、すこし後ならできるけど。

3. 鈴木さん、今日はお目にかかれて、うれしかったです

  • 1.こちらこそ、またお会いしたいです。
  • 2.私も楽しみにしております。
  • 3.では、どうぞ、ご覧ください。

4. りんさん、日本語ずいぶんうまくなってきたし、スピーチ大会に出てみたら

  • 1.じゃあ、寝るのはあきらめます。
  • 2.はい、私も出てみたいと思ってるんです。
  • 3.出てみたらいかがですか。

5. あっ、松本君、シャツのボタンが取れそうだよ

  • 1.どこで落としちゃったんだろう。
  • 2.ほんとだ。ぜんぜん気がつかなかったよ。
  • 3.拾ってくれたの。ありがとう。

6. 先輩、先、いただいた資料のことで、お聞きしたいことがあるんですが

  • 1.どこで聞いたの。
  • 2.どの資料がほしいの。
  • 3.うん、どんなこと。

7. お料理、冷めないうちに、どうぞ召し上がってください

  • 1.はい、では、さっそくいただきます。
  • 2.では、冷めてからいただくんですね。
  • 3.冷たいうちに、いただきます。

8. あっ、アパートの部屋の鍵、かけずに出てきちゃったかもしれない

  • 1.部屋に忘れ物しちゃったの。
  • 2.帰って確かめたほうがいいんじゃない。
  • 3.かけてきたんなら、大丈夫だね。

9. 棚の片付け、まだ終ってないみたいだけど。遅いし、それ、明日にしてもかまわないよ

  • 1.ああ、今日中に終らせないといけないんですね。
  • 2.今日はちょっとお手伝いできないんですが。
  • 3.すみません、じゃ、今日はここまでにします。
