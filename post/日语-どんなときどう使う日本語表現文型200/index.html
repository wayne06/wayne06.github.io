1. 時間

  1. ～とき

  2. Ｖながら

  3. ～前に

  4. ～後で・～後

  5. Ｖてから

    • a　手を洗ってから食べます。

    • b　日本に来てからずっとこの寮に住んでいます。

  6. Ｖるところです・Ｖているところです・Ｖたところです

  7. Ｖたばかりです

  8. ～間・～間に

2. 比較・対比

  1. Ｎ1は～が、Ｎ2は

  2. Ｎ1はＮ2より

  3. Ｎ1よりＮ2のほう

  4. Ｎ1とＮ2とどちら

  5. Ｎ1はＮ2ほど～ません

  6. Ｎ1の中でＮ2がいちばん

  7. ～すぎます

3. Ⅰ　並列

  1. Ｖ1たりＶ2たりします

    • a　本を読んだりテレビを見たりします。

    • b　出たり入ったりしています。

  2. ～たり～たりします

  3. Ｎ1も～し、Ｎ2も

3. Ⅱ　経験

  1. Ｖたことがあります

  2. Ｖることがあります・Ｖないことがあります

4. Ⅰ　原因・理由

  1. ～から

  2. ～ので

  3. ～て・～で・～なくて

  4. ～し

  5. ～ため（に）

4. Ⅱ　逆接

  1. ～が

  2. ～のに

5. Ⅰ　希望

  1. Ｎがほしいです

  2. Ｖたいです

  3. ～がります

  4. ～といいです・～たらいいです・～ばいいです

5. Ⅱ　目的

  1. ～に（行きます）

  2. Ｖるため（に）・Ｎのため（に）

  3. Ｖるのに・Ｎに

  4. Ｎのため（に）

6. 可能・難易

  1. Ｖることができます・Ｎができます

  2. 可能動詞：読める・食べられる

  3. 可能の意味のある動詞：見える・聞こえる

  4. Ｖやすいです・Ｖにくいです

  5. 自発：思われる・感じられる

7. 依頼・命令

  1. Ｎをください・Ｎをくれませんか・*Ｎをくださいませんか

  2. Ｖてください・Ｖてくれませんか・*Ｖてくださいませんか・
    Ｖないでください・Ｖないでくれませんか・*Ｖないでくださいませんか

  3. Ｎをもらえませんか・Ｎをいただけませんか

  4. Ｖてもらえませんか・Ｖていただけませんか・
    Ｖないでもらえませんか・Ｖないでいただけませんか

  5. おＶください

  6. Ｖ（さ）せてください

  7. 命令形：止まれ・食べろ

  8. Ｖるな

  9. Ｖなさい

8. Ⅰ　勧誘・申し出

  1. Ｖましょう・Ｖよう

  2. Ｖませんか

    • a　一緒に行きませんか。

    • b　うちへ来ませんか。

  3. Ｖましょうか

    • a　コーヒーを飲みましょうか。

    • b　手伝いましょうか。

8. Ⅱ　提案

  1. Ｖたほうがいいです・Ｖないほうがいいです

  2. Ｖたらどうですか

  3. 疑問詞＋Ｖたらいいですか／Ｖばいいですか

  4. Ｖるといいです・Ｖたらいいです・Ｖばいいです

9. 許可・禁止・義務・不必要

  1. Ｖてもいいです・Ｖてもかまいません

  2. ～てもいいです・～てもかまいません・
    ～でもいいです・～でもかまいません

  3. ～てはいけません・～ではいけません

  4. ～なければなりません・～なければいけません・
    ～なくてはなりません・～なくてはいけません

  5. ～Ｖなくてもいいです・～Ｖなくてもかまいません

  6. ～なくてもいいです・～なくてもかまいません

10. 変化

  1. イＡくなります・～になります

  2. イＡくします・～にします

  3. Ｖるようになります・Ｖなくなります

  4. Ｖてきます・Ｖていきます

11. Ⅰ　伝聞・引用

  1. ～そうです

  2. 「…」と（言います）ｋ・…と（言います）

  3. ～ように（と）（言います）

  4. Ｎ1というＮ2

  5. 命令形と（言います）・禁止形と（言います）

11. Ⅱ　説明

  1. ～のです

  2. ～のは～です

12. Ⅰ　意志

  1. Ｖよう

  2. Ｖようと思います

  3. Ｖるつもりです・Ｖないつもりです

  4. Ｖようとします

  5. Ｖるようにします・Ｖないようにします

  6. Ｖようとしません

12. Ⅱ　決定

  1. Ｎにします

  2. Ｎになります

  3. Ｖることにします・Ｖないことにします

  4. Ｖることになります・Ｖないことになります

13. Ⅰ　意図的行動

  1. Ｖてみます

  2. Ｖてしまいます

    • a　全部読んでしまいます。

    • b　忘れてしまいました。

  3. Ｖておきます・Ｖないでおきます

13. Ⅱ　開始・終了

  1. Ｖはじめます

  2. Ｖつづけます

  3. Ｖおわります

  4. Ｖだします

14. Ⅰ　様子・状態

  1. Ｖています

  2. Ｖてあります

  3. ～そうです

    • a　このケーキはおいしそうです。

    • b　ボタンがとれそうです。

    • c　今日は早く帰れそうです。

  4. Ｎ1はＮ2が

  5. Ｎがします

  6. Ｎをしています

  7. ～まま

14. Ⅱ　比況・例示

  1. Ｎのようです・Ｎみたいです

  2. Ｎらしいです

  3. Ｎ1のようなＮ2・Ｎのように・
    Ｎ1みたいなＮ2・Ｎみたいに

15. 推量

  1. ～でしょう

    • a　明日はいい天気でしょう。

    • b　これ、見て。いい色でしょう。

  2. ～だろう

    • a　今日、中川氏は会に出席しないだろう。

    • b　これ、いいだろう。きのう買ったんだよ。

  3. ～だろうと思います

  4. ～らしいです

  5. ～かもしれません

  6. ～はずです

    • a　4時に着くはずです。

    • b　寒いはずですよ。0度ですから。

  7. ～ようです・～みたいです

16. 条件

  1. ～たら

  2. ～ば・～なら（ば）

  3. ～と

  4. Ｖたら

  5. ～ても・～でも

  6. ～なら

  7. Ｖると～た・Ｖたら～た

    • a　会場に着くと、コンサートはもう始まっていました。

    • b　「ボン」と呼ぶと、その犬はこちらを見ました。

    • c　彼は部屋に入ると、すぐ窓を開けました。

  8. Ｖていると～た・Ｖていたら～た

17. 授受

  1. Ｎをあげます・Ｎをさしあげます・Ｎをやります

  2. Ｎをもらいます・Ｎをいただきます

  3. Ｎをくれます・Ｎをくださいます

  4. Ｖてあげます・Ｖてさしあげます・Ｖてやります

  5. Ｖてもらいます・Ｖていただきます

  6. Ｖてくれます・Ｖてくださいます

18. 使役

  1. Ｖ（さ）せます

    • a　強制：子供に部屋を掃除させます。

    • b　許可・恩恵：子供にアイスクリームを食べさせます。

    • c　誘発：友達を笑わせます。

  2. Ｖ（さ）せてくれませんか・（さ）せてくださいませんか・
    Ｖ（さ）せてもらえませんか・Ｖ（さ）せていただけませんか

19. 受身・使役受身

  1. Ｖ（ら）れます

    • a　基本的な受け身：母にしかられます。

    • b　持ち主の受け身：人に足をふまれます。

    • c　被害の受け身：子供に泣かれます。

    • d　非情の受け身：試合が行われます。

  2. Ｖさ（せら）れます：使役受身

20. 敬意

  1. おＶになります

  2. Ｖ（ら）れます

  3. 尊敬を表す特別な動詞：おっしゃいます

  4. おＶください

  5. おＶします・おＶいたします

  6. 謙譲を表す特別な動詞：申し上げます

  7. 丁寧な気持ちを表す特別な動詞：ございます

  8. お～・ご～

  9. おＶです
