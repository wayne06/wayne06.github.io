N3语法07 - 传闻

Posted by WZhong on Thursday, September 24, 2020

TOC

  • ～んだって： 听说（偏口语表达）
  • ～だそうです： 听说（直接信息来源，可信度高）
  • ～らしい： 听说（间接信息来源，可信度低）
  • ～ということだ： 听说（偏正式表达）
  • ～とのことだ： 听说（转述口述内容，是「と言ってました」的正式说法）
  • ～とか：
  • ～という：
  • ～と言われている：
  • 题目

普通型＋そうです：表示传闻
雨が降るそうです。
ます型＋そうです：表示样态
雨が降りそうです。

～んだって： 听说（偏口语表达）

～だそうです： 听说（直接信息来源，可信度高）

～らしい： 听说（间接信息来源，可信度低）

～ということだ： 听说（偏正式表达）

～とのことだ： 听说（转述口述内容，是「と言ってました」的正式说法）

～とか：

～という：

～と言われている：

  1. 【】

    • 彼は最近新しい彼女ができたんだって。
    • 天気予報によると、明日雨が降るんだって。
    • 佐藤さんの奥さんは英語の先生なんだって？
    • あの店のケーキ、おいしいんだって？
    • 山田さん、お酒が嫌いなんだって？
    • あの二人は付き合ってるんだって？
    • 店の人の話では、この地方の米はとても美味しいということだ。（ちほう）（こめ）
    • 市の知らせによれば、この道は来週から工事が始まるということだ。（こうじ）
    • 山田さんは熱があるので、明日はいけないとのことだ。
    • 課長は君に話があるからすぐ来て欲しいとのことだ。
    • 大島さん、今日休むって。
    • 彼はすぐ来るって。
    • なんて？なんだと？なんですって？
    • 山田さんは今日風邪で休むとか。
    • 本人に聞いたわけではないが、あの二人が離婚したとか。（りこん）
    • 山田さんなら途中で事故があったとか。
    • あの店のパンはとても美味しいとか。
    • この祭りは村で古くから行われたという。（まつり）（ふるく）（おこなわれた）
    • 豆腐は1300年ぐらい前に中国から日本に伝わったという。（つたわった）
    • 納豆は体にいいと言われている。（なっとう）
    • あの会社はもうすぐ倒産すると言われている。
    • 日本人は働きすぎと言われている。

题目

  • 1　パソコンがこわれてしまったために、（）。

    • a　新しいのを買おう
    • b　資料が作れなかった（つくれ）
    • c　直してくれませんか（なおして）

  • 2　台風15号によって（）。（たいふう）

    • a　橋が流された（はし）（ながされた）
    • b　明日は大雨だろう（あす）
    • c　明日は外出したくない

  • 3　高い技術（）詳しい健康チェックができるようになった。（ぎじゅつ）（くわしい）（けんこう）

    • a　によって
    • b　によれば
    • c　によると

  • 4　小さな不注意（）大問題がおこることもある。（ふちゅうい）（だいもんだい）

    • a　から
    • b　まで
    • c　には

  • 5　自転車の事故が増えたことから、（）。（じこ）（ふえた）

    • a　気をつけよう
    • b　自転車には乗りたくない
    • c　警察の注意が厳しくなった（けいさつ）（ちゅうい）（きびしく）

  • 6　弟のせいで（）。

    • a　楽しかった
    • b　よく遊べた（あそべた）
    • c　母にしかられた

  • 7　この薬（）病気を治すことができた。（なおす）

    • a　のおかげで
    • b　のせいで
    • c　から

  • 8　やっと運転免許が取れたんだから、（）。（うんてんめんきょ）

    • a　車を買った
    • b　車を買いたい
    • c　車は買わなかった

  • 9　先生、すみません。かぜをひいて（）、今日は休ませてください。

    • a　しまったので
    • b　しまったんですから
    • c　しまって

  • 10　今年は梅雨に雨の量が少なかった（）、米や野菜などがよく育っていない。（ことし）（つゆ）（りょう）（すくなかった）（こめややさい）（そだって）

    • a　おかげで
    • b　ために
    • c　のなら

  • 11　今日は（）、早く家に帰りたい。

    • a　疲れたせいで
    • b　疲れたのでは
    • c　疲れたから

  • 12　あなたは（）どんな仕事でもするんですか。

    • a　給料さえ高ければ
    • b　給料が高いのでは
    • c　給料が高いのだから

  • 13　せっかく京都まで（）、京都にしかない物を食べませんか。

    • a　来たのでは
    • b　来たことから
    • c　来たんだから

  • 14　ひろしは絵を（）、デザインの仕事に興味を持ったようだ。

    • a　ほめられたのだから
    • b　ほめられたことから
    • c　ほめられさえすれば

  • 15　太陽熱（）発電は、24時間可能だ。（たいようねつ）（はつでん）（かのう）

    • a　から
    • b　による
    • c　のための

  • 16　オートバイを（）、今、お金をためてます。

    • a　買いたいなら
    • b　買いたいので
    • c　買いたいのでは

  • 17　ホームステイに（）、早く申し込みをしたほうがいいですよ。

    • a　参加すれば
    • b　参加するため
    • c　参加するなら

  • 18　そんなにほめ言葉を（）、かえって恥ずかしいです。（はずかしい）

    • a　言われては
    • b　言われたのだから
    • c　言われさえすれば

  • 19　たとえどんなにいい服を（）着る機会がなかったら意味がない。

    • a　買ったのだから
    • b　買っても
    • c　買ったのでは

  • 20　（）、「ズボン」はフランス語から来た言葉だということです。（ことば）

    • a　先生の説明は
    • b　先生の説明では
    • c　先生の説明からは

  • 21　今朝の新聞（）、痛み止めの新しい薬が発売されるということだ。（いたみどめ）（はつばい）

    • a　によって
    • b　によったら
    • c　によると

  • 22　（）、この家は300年ぐらい前に建てられたと言われている。

    • a　確かではないが
    • b　山川さんの話では
    • c　林さんから聞いたのだか

  • 23　今、テレビの天気予報で見たんだけど、あしたは全国的に雨だ（）よ。（ぜんこくてき）

    • a　と聞いている
    • b　と言われている
    • c　そうだ

  • 24　足の裏を日に当てると健康に（）とか。本当だろうか。（あし）（うら）（あてると）

    • a　いい
    • b　いいです
    • c　いいそうだ

  • 25　「あしたは（）。」
    　　「わあ。いやだなあ。あしたは野球の練習があるんだ。」

    • a　暑いと
    • b　暑いって
    • c　暑くって

  • 26　昔、この地方には、珍しい習慣が（）という。（ちほう）（めずらしい）（しゅうかん）

    • a　あった
    • b　あったそうだ
    • c　あったんだって

  • 27　この地域の土地の値段は今後もあまり高く（）という。（ちいき）（とち）

    • a　なりません
    • b　ならないでしょう
    • c　ならないだろう
