0. フローチャート

  • 0.　スタート

  • 1.　販促トップ　（顧客管理のトップページから販促のアイコンをクリックして、販促活動の設定を始める。）

  • 2.　媒体を選択　（はがき、タックシールなどから販促媒体を選ぶ。）

  • 3.　顧客情報検索　（最終購買日、累計購買回数、累計購買金額などで優良顧客を選択する。）

  • 4.　検索結果チェック

    • if 検索結果<=30

      • 4.1　画面に出力する　（該当するデータが検索結果画面に表示される。） ⇒ 5

    • if 検索結果>30

      • 4.2　絞り込むか確認　（検索結果が30より多い時は、確認画面が現れ、さらに絞り込むか選択できる。）

        • if 絞り込む　（「はい」ボタンを押すと、親画面に戻って検索条件を追加できる。）　⇒ 3

        • if 絞り込まない　⇒ 4.1

  • 5.　出力

  • 6.　エンド

1. 目標

  • 仕様書の顧客管理に関する語彙が理解できる

  • 販促活動の対象を選ぶ処理が理解できる

2. 表現の理解

  • 顧客管理のトップページから販促のアイコンをクリックして、販促活動の設定を始める。

  • はがき、タックシールなどから販促媒体を選ぶ。

  • 最終購買日、累計購買回数、累計購買金額などで優良顧客を選択する。

  • 検索結果が30より多い時は、確認画面が現れ、さらに絞り込むか選択できる。

  • 「はい」ボタンを押すと、親画面に戻って検索条件を追加できる。

  • 該当するデータが検索結果画面に表示される。
