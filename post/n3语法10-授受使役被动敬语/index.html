授受关系

～てあげる　～をあげる
～てくれる
～てもらう

不关我事
あげます：花さんは　トニさんに　プレゼントを　あげます。
もらいます：トニさんは　花さんに　プレゼントを　もらいます。

关我事
私は　トニさんに　プレゼントを　あげます。
トニさんは　私に　プレゼントを　くれます。
私は　花さんに　プレゼントを　もらいます。

step1 step2 step3 get
和我有关 我是施 （私は）
和我有关 我是受 施做主语
和我有关 我是受 受做主语
和我无关 施做主语
和我无关 受做主语

  1. 「私」不只是我， 也包括“我方人员”。

    • X 山田さんは母に服を作ってあげました。
    • V 母は山田さんに服を作ってもらいました。（つくって）

  2. 收到好东西（心存感激） 才可以用此表达。

    • X 山田さんに請求書をあげました。
    • V 山田さんに請求書を渡しました。（わたし）
    • X 先生はたくさん宿題をくれました。
    • V 先生はたくさん宿題を出しました。
    • 泥棒に荷物を持っていかれました。（どろぼう）
    • トニさんに荷物を外に持って行ってもらった。（そとにもって）

  3. 从组织机构得到某物， ～から～をもらう

    • 図書館からパンフレットをもらった。

  4. 敬语形态

    あげる　―　さしあげる
    くれる　―　くださる（くださいます）
    もらう　―　いただく（いただきます）

    • 私は田中先生に花をさしあげました。
    • 田中先生は私に手紙をくださいました。
    • 私は田中先生に手紙をいただきました。

  5. ～てあげる・～てさしあげる　【我帮你…，（高高在上，很少对尊长使用）】

    • ビーマンが嫌いなの？じゃあ私が食べてあげる。

  6. ～やる・～やります　【身份低的人、物体、宠物】

    • 花に水をやる
    • 犬に餌をやる（えさ）

  7. 借りる・貸す（かりる・かす）【借】

    • 花さんはトニさんに漫画を貸してあげる。
    • 花さんは私に漫画を貸してくれる。
    • 私は花さんに漫画を貸してもらう。
    • 私は花さんに漫画を借りる。

尊敬语（尊他）

和对方相关的任何事物
称呼：～先生、～さん、～様、～部長、どなた、あの方
形容词：お忙しい（おいそがしい）、ご綺麗（ごきれい）、お元気
名词：お名前、お仕事、お電話
动词：ご協力（ごきょうりょく）、ご希望（ごきぼう）、ご配慮（ごはいりょ）

自谦：わたくし、いたす、弊社（へいしゃ）、拝見（はいけん） - お仕事は何ですか。 - あちらは山田様です。 - 先生は来週もお忙しいようです。 - 先生は来週もご多忙のようです。（たぼう） - ご家族はどちらにいらっしゃいますか。

敬语 文型 例文
尊敬语 動一　＋　（ら）れる 先生、新聞を読まれますか。
お/ご　＋　名词、形容词、形容动词 お名前、ご主人、お忙しい、ご立派
お/ご　＋　動二　+　になる/なさる/くださる お読みになります。
お/ご　＋　漢语　+　になる/なさる/くださる お使いになります。
ご出席になります。
ご検討ください。
ご利用なさいますか？
ご覧になりましたか？
特殊形 （见下图表格）
谦让语 お/ご　＋　する/いただく お願いします。
ご紹介します。
お待ちいただきます。
させていただく 変更させていただきます。
特殊形 （见下图表格）
丁重语 おる・参る・申す… 社内におります。
丁宁语 语尾为「です」「ます」「ございます」 そうです。
行きます。
おはようございます。
美化语 お/ご～ お弁当・お茶・お酒

题目

最も適当なものを選びなさい。（もっとも）（てきとう）（えらび）

  • 1　昨日かぜで一日中寝ていた。同じアパートのカンさんが、晩ご飯を作って（a.あげた　b.もらった　c.くれた）。カンさんは料理が上手で、ときどき大家さんにも国の料理を作って（a.あげている　b.もらっている　c.くれている）。「ぼくは大切な仕事がある日は、大家さんに頼んで（a.起こしてあげて　b.起こしてもらって　c.起こしてくれて）いるので、そのお礼なんです。」とカンさんは言う。

  • 2　隣の鈴木さんは親切な人だ。私は前に鈴木さんにこわれた自転車を（a.直してあげた　b.直してもらった　c.直してくれた）ことがある。その自転車は、学校の（a.先輩にゆずってあげた　b.先輩にゆずってもらった　c.先輩がゆずってあげた）物だ。鈴木さんがこわれていたベルを（a.取り替えてあげた　b.取り替えてもらった　c.取り替えてくれた）ので、ずっと使っていたのだが、この間、だれかにそのベルを（a.盗んでもらって　b.盗ませて　c.盗まれて）しまった。

  • 3　ある日、宿題で書いた日本語の作文をだれかに（a.見てあげたい　b.見てもらいたい　c.見せてもらいたい）と思ったので、アルバイトをしている店の店長に頼んでみた。すると、高校生の娘さんを（a.紹介してあげた　b.紹介してもらった　c.紹介してくれた）。娘さんは英語が苦手らしいので、私はときどき日本語を（a.教えてあげる　b.教えてもらう　c.教えてくれる）かわりに、英語を（a.教えてあげる　b.教えてもらう　c.教えてくれる）ことにした。

（）の中の動詞を文章の流れに合う形にして、書きなさい。（どうし）（ながれに）

  • 1　父の店がテレビで（紹介する　=>　）たので、急に客が増えた。そのため、父は兄や姉にも店を（手伝う　=>　）ている。しかし、私には（手伝う　=>　）つもりはないらしい。

  • 2　サッカー部に入ったが、なかなかボールに（触る　=>　）てもらえず、先輩に（走る　=>　）てばかりいる。つまらないのでゆっくり走ると、すぐに大きな声で（注意する　=>　）て、練習の後も部屋の掃除を（する　=>　）。もう辞めようかと思った。でも、先日試合に（出場する　=>　）てもらって、先輩によくがんばったと（）たので、もう少し続けてみようと思う。

  • 3　急に昔の友達に会いたくなったので、メールを（送る　=>　）た。冗談を言ってよく私たちを（笑う　=>　）ていた人だ。会って話を聞くと今は3人の子供を（育てる　=>　）ているそうだが、明るさは昔と変わっていない。あまり厳しいことを（言う　=>　）ないで、子供たちを自由に（遊ぶ　=>　）ているのだそうだ。彼女らしい（育てる　=>　）方だと思った。

  • 4　病院や銀行などでは、順番を待つ人に待ち時間を長く（感じる　=>　）ないように、いろいろな工夫を（する　=>　）ている。待合室などに（置く　=>　）てあるテレビや雑誌もその一つだ。絵本やおもちゃも、親に連れて（くる　=>　）た子供を（あきる　=>　）ないためのものだ。しかし、一番いいのは、病院や銀行が人を（待つ　=>　）ないことだろう。

最も適当なものを選びなさい。（もっとも）（てきとう）（えらび）

  • 1　高橋先生が先週水曜日に帰国（）。（たかはし）（きこく）

    • a　いたしました
    • b　されました
    • c　いたされました

  • 2　客「あの、私の番はまだでしょうか。20分ぐらい待っているんですけど。」
    　銀行員「何番の紙を（）。ああ、12番ですね。では、次です。」

    • a　お持ちされますか
    • b　お持ちですか
    • c　お持ちしていますか

  • 3　お父様はあなたに何と（）。

    • a　申されましたか
    • b　申し上げましたか
    • c　おっしゃいましたか

  • 4　私はリーと（）。台湾からの留学生です。

    • a　申します
    • b　申し上げます
    • c　おっしゃいます

  • 5　客「すみません。牛乳はどこですか。」
    　店員「あ、牛乳ですか。牛乳はあそこのパン売り場のとなりに（）。」

    • a　いたします
    • b　ございます
    • c　おります

  • 6　どうぞご自由パンフレットを（）ください。

    • a　お取り
    • b　いただいて
    • c　お持ちして

  • 7　お忙しいとは（）が、できるだけ早くお返事をいただけますか。

    • a　ご存じます
    • b　ご存じです
    • c　存じます

  • 8　では次に、こちらのグラフを（）。

    • a　ごらんください
    • b　ごらんでください
    • c　ごらんして下さい

  • 9　来月国から両親が（）。

    • a　見えます
    • b　参ります
    • c　おいでです
